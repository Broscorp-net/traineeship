Java Development Kit (JDK) -(для программирование) набор для разработки на java, включает в себя набор стандартных библиотек, компилятор, JRE, JVM.
Java Runtime Environment(JRE) -(для роботы) минимальный набор для запуска программы написанной на java. не содержит копилятора и отладчика, состоит из стандартных библиотек java и JVM.
Java Virtual Machine (JVM) - Основная часть JRE - позволяет использовать java байт код и является ключем в реализации коросплатформености в java. так как скомпелированый единожды байт код можно передавать на разные JVM (в зависимости от платформы) и он будет на них исполнятся.
JIT - инструмент JVM для ускорения работы с байт кодом.
