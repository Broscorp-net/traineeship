Все начинается с исходного кода(.java) -> которым питается JDK, компилируя исходный код в байт код(.class) -> которым питается JVM, преобразовывая .class в машинный код.


JRE - java rnutime enviromet, является частью JDK,состоит из стандартного  набора библиотек Java и минимальной реализации JVM, позволяет запускать Java приложения без компиляторов и других средств разработки.


JIT - just in time компилятор, является частью JVM, преобразовывает байт код в машинный код во время исполнения программы.



JDK - java development kit, инструменты разработчика необходимые для написания Java приложений.


JVM - java virtual machine, виртуальная машина джава исполняет байт код, который предварительно был создан в JDK компилятором.
